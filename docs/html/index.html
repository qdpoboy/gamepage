像素风格个人站原型图

这是一套基于像素风格设计的个人站原型图。每个页面都采用了复古游戏风格的设计元素，并融入了现代的交互体验。

设计理念

采用8-bit/16-bit像素画风，结合现代网页设计理念，打造独特的游戏化浏览体验。暗色主题为主，搭配鲜明的强调色，突出内容的同时保持视觉舒适度。

交互说明

所有交互都模拟复古游戏的操作方式，包括选择菜单、对话框、收集要素等。加入了声音反馈和像素动画，增强沉浸感。

页面说明

原型包含5个主要页面：首页（世界地图）、分类（卡带仓库）、技能树（能力神殿）、心得（智慧书馆）和个人介绍（勇者之路）。每个页面都有其独特的游戏化设计。

使用说明

点击左侧导航可以查看不同页面的原型设计。每个原型页面都包含了详细的交互说明和设计注释。
